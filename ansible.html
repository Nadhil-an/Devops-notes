Cheat Sheet

Commands worth having on the tip of your tongue.

CommandWhat it does
ansible all -m pingCheck connectivity to all hosts in inventory
ansible-playbook site.ymlRun a playbook against the inventory
ansible-playbook site.yml --checkDry run — show what would change, without changing anything
ansible-playbook site.yml -i hostsRun using a specific inventory file
ansible-doc <module>Show documentation for a specific module
ansible-galaxy install <role>Install a shared role from Ansible Galaxy
