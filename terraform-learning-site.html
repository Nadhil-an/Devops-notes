Cheat Sheet

Commands worth having on the tip of your tongue.

CommandWhat it does
terraform initDownloads providers, sets up the working directory
terraform planShows what changes would be made, without applying them
terraform applyApplies the changes to actually create/update infrastructure
terraform destroyTears down everything Terraform created
terraform fmtAuto-formats .tf files to standard style
terraform validateChecks the config syntax without contacting the cloud provider
terraform state listLists all resources currently tracked in state
