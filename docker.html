Cheat Sheet

The commands worth having on the tip of your tongue.

CommandWhat it does
docker build -t name .Build an image from a Dockerfile
docker run -d -p 8080:80 nameRun a container in the background with a port mapped
docker ps / docker ps -aList running / all containers
docker stop <id>Gracefully stop a container
docker rm <id>Remove a stopped container
docker rmi <image>Remove an image
docker exec -it <id> bashOpen a shell inside a running container
docker logs -f <id>Follow live logs
docker pull / docker pushDownload / upload image from a registry
docker volume lsList volumes
docker network lsList networks
docker compose up -dStart all services in the background
docker compose downStop and remove everything from Compose
docker system pruneClean up unused images/containers/networks
